Voice and Tone

Easy Tiger is a friendly local shop, and we convey our down-to-earth vibe through the voice and tone of our copy.

General Copy

When talking about upcoming events, use a casual, fun voice, as if speaking to a friend.

Example

“We've got ping pong tables out back. Talk about full service!”

About Us / Item Descriptions

When talking about our business we use clear, concise verbiage.

Example

“Easy Tiger features artisan bread (including baguettes, croissants, pretzels), locally roasted coffee, housemade meats (sausages, corned beef, pastrami), a full bar and more than 30 local and boutique draft beers.”

Markup

We use Sass to keep our code as clean and organized as possible. Below are some examples of code used on the site.

Grid Mixin

Our site is built on a custom, 12 column grid base.


$tot-width: 100%;
$tot-num-col: 12;
$gut-width: 10%;
$col-width: ($tot-width - (($tot-num-col - 1)*$gut-width))/$tot-num-col;

@mixin grid($col-num) {
	float: left;
	width: ($col-width*$col-num)+(($col-num - 1)*$gut-width);
	margin-right: $gut-width;

	&:last-child {
		margin-right: 0;
	}
}

Rem Conversion Mixin

We use a mixin to convert to pixels and provide a backup for browsers that don't support rem.


@mixin fntz($rem){
	size: (($rem*16) + px);
	size: ($rem + rem);
}

Color Variables

We use variables to define color so we can easily make changes at any time.


$color-text: #2e2c2c;
$color-primary: #f6ad22;
$color-secondary: #c37012;
$color-tertiary: #6b804f;
$color-dark: #544834;
$color-light: #f5f3ec;

Font Variables

We import Google fonts into a Sass partial and use variables to define type styles.


@import url(https://fonts.googleapis.com/css?family=Alegreya:900,700|Lato:400,700);

$font-headings: "Alegreya", serif;
$font-main: "Lato", sans-serif;

Typography

Our typography reflects the friendly feeling of our shop. Headings are bold and chunky with a pub type feel. Body text is simple and readable. Generous spacing is provided for cleanliness and legibility.

Headings

Headings and subheadings are set in Alegreya. Horizontal rules are used sparingly for heirarchy.

  • Heading 1

    Alegreya Bold 700 2.75rem/48px

  • Heading 2

    Alegreya Bold 700 1.75rem/28px

  • Body copy

    Lato Normal 400 1rem/16px

Text Elements

All body text is set in Lato with a line height of 1.6em/26px.

Example

Easy Tiger features artisan bread (including baguettes, croissants, pretzels), locally roasted coffee, housemade meats (sausages, corned beef, pastrami), a full bar and more than 30 local and boutique draft beers.

Images

Most of the look and feel of our site is achieved through typography and color; images are used sparingly throughout the site.

Logo

Our logo is one solid color, only displayed in white or very dark gray, and always with plenty of contrast to ensure legibility.

Example

Icons

Like our logo, icons are simple and one color. Icon links are displayed at 75% opacity, changing to 100% on hover.
